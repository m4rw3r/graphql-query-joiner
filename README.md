## Packages

### GraphQL AST Client

GraphQL client based on parsed query ASTs with a simple API.

## Development

Requirements:

- [PNPM](https://pnpm.io/)

```bash
pnpm install

# Typecheck TypeScript in all packages
pnpm check

# Run tests for all packages
pnpm test

# Lint all packages
pnpm lint

# Format code in all packages using Prettier
pnpm format

# Build all packages
pnpm build
```
